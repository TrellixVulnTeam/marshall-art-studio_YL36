Account Management

Account management requires you to create an account before you start creating work items. This allows for work items to be grouped under an account and for the data associated with an account to be copied to the work item.

After account management is turned on, create all work items from an account or with an account reference. By default, work items can only be created by first creating an account and then creating the work items for that account.

What is an Account?

An account is a type of work item that contains most of the features of a work item. Account transactions use the same product definition files that a regular line of business (LOB) work item uses. They can also be maintained using the toolkit.

Account can also use work lists to view the list of accounts and search the accounts available in the AgencyPortal application.

Configuring Account Management

Set the following properties to start using the account management feature:

application.account_management=true This turns on account management for the entire application.
application.account_management_exit_next_page=/DisplayWorkInProgress?WORKITEMID=${ACCOUNTID}&action=View&WorkListType=AccountsView Set this property when using the template support for account management. When a user clicks Save and Exit, they are taken to the Account Detail view of the account associated with the work item.

Account Management Infrastructure

The account management infrastructure is supported by two basic interfaces:

  • one that models the basic contents of an account
  • one that models the mechanism for reading and writing the account to the database

These interfaces are the following:

  • com.agencyport.account.model.IAccount
  • com.agencyport.workItem.model.IWorkItemManager

The application properties used to identify the implementations to the factory are the following:

  • account.classname
  • account.manager_classname

The product's canonical implementations of these interfaces are the following:

  • com.agencyport.workitem.impl.Account
  • com.agencyport.account.impl.AccountManager

These are engaged by the default if the properties are not otherwise explicitly configured.

Personal and Commercial Account Templates

Two account templates are provided:

  • Personal Account
  • Commercial Account

These templates provide the basic fields a user needs to fill out to set up either a personal line or commercial line account. You need at least one of these templates to get account management up and running. Contact Agencyport for access to these templates.
