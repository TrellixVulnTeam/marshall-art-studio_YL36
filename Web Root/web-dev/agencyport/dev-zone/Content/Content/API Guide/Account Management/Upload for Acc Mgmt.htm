Upload for Account Management - Refer to Turnstile

AgencyPotal includes support for uploads to accounts as part of our Account Management feature.

When Account Management is turned on and an upload is submitted to AgencyPortal, a check is performed to either the social security (SSN) or tax id number (FEIN) on the upload to see if it matches the SSN or FEIN on any existing accounts. If only one matching account is found, the following window displays:

[NEED 5.0 SCREEN SHOT]

After the uploaded work item is committed (first page is processed), the work item is added to the matched account.

If the user uploads to AgencyPortal and multiple matching accounts are found, the following window displays, allowing them to choose one of the matching accounts:

[NEED 5.0 SCREEN SHOT]

After selecting the correct account and clicking OK, the following window displays over the work item.

[NEED 5.0 SCREEN SHOT]

After the upload work item is committed (first page processed), the work item is added to the selected account.

If the user uploads to AgencyPortal and no matching accounts are found, the following window displays, allowing them to search for an account or create a new account:

[NEED 5.0 SCREEN SHOT]

If the search returns more than one account with the same name, the account ID is listed to help differentiate the accounts.
