Dashboard

Dashboard reporting establishes a standard framework for delivering both summary chart based and detail listing reports to the AgencyPortal user community. Its use is entirely option for migrating previous release level applications, but is certainly a feature you should seriously consider in either context.

The underlying architecture of the dashboard uses a graphics component AM charts (http://www.amcharts.com/) and Adobe Flash component (http://www.adobe.com/) for rendering graph-based reports.
