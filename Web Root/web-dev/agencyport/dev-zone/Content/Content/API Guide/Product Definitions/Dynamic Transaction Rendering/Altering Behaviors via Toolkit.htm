Altering Behaviors via Toolkit

If the behavior action is Alter, an additional Add 'alter' property button displays. Perform the following to alter a behavior via toolkit:

  1. Click the Add 'alter' property button to allow the user to select from a list of available properties to alter. A dialog box displays with fields relevant to that type of property:
  2. Select the alter property type (i.e., format mask). The palette changes to the Alter Properties tab and a new entry is added to represent the selected property type.
  3. Select a valid alter properties. The following is a list of valid alter properties:
    Available 'Alter' BehaviorsPurpose
    CorrectionAlter corrections on a field element.
    Custom RosterReader ClassSets or overrides the custom roster reader class name attribute on a roster page element.
    Data FilterSets or overrides the data filter attribute on a roster page element.
    Default ValueSets the default value of a field based on a defined condition.
    Empty Roster MessageSets or overrides the empty roster message element on a roster page element.
    Enable RecoverSets the enable recover attribute on the roster page element to true or false.
    Format MaskAdds a mask over the field to give the user the pattern of the data that needs to be entered.
    Id or Group IdSubstitutes a replacement parameter within an ACORD path with the precondition value of that name.
    Help TextUpdates an existing or creates a new balloon helper with the element text for the given field element.
    JavascriptApplies the java script reference to the page element specified in the statement.
    LabelApplies the element text to the label of the field element.
    LegendApplies the element text value to the legend of the page element specified in the statement.
    LengthUpdates the length property on the text based field element to the element value.
    Max. EntriesSets or overrides the max entries attribute on a roster page element.
    Max. LengthUpdates the maxLength property on the text based field element to the element value.
    MessageApplies the element text as the message text of the field element.
    Message Class FilterApplies the new value as the message class filter of a given connector or all connectors.
    Min. EntriesSets or overrides the min entries attribute on a roster page element.
    Min. LengthUpdates the minLength property on the text based field element to the element value.
    OptionlistReferenceTagApplies the option list tag to the selected list field element.
    ReadonlySets a field to a readonly state based on a defined condition.
    RequiredSets a field to required based on a defined condition.
    Style ClassChanges the CSS style class for a field.
    Suppress Message DisplayIf true, will set or override the display of messages on the screen produced by a connector.
    TitleThis is the display name for the behavior.
    ValidationUpdates an existing validation element or adds a new one of the given type for the given field .
    ValueSet the value of a field based on a certain condition. This value must be entered without quotes.
  4. A dialog box specific to that type displays with the necessary fields.
  5. Click Apply. The dialog box closes.
