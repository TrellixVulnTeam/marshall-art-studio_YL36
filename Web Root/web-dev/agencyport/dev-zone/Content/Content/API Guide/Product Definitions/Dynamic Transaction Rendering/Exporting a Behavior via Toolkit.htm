Exporting a Behavior via Toolkit

Perform the following to export a behavior via toolkit (such as, view or share the xml produced from a group of behaviors or edit the transaction outside of toolkit and re-import it):

  1. Select the group of behaviors in the navigation.
  2. Click the Export action in the editor. A pop-up window displays with options to open or save the file.
  3. Select an option and click OK.
