Adding an Option List via Toolkit

Perform the following to add an option list via toolkit:

  1. Select a newly added static option list collection under the Lists folder in the navigation.
  2. Click the Add Option List action.
  3. Click on the new option list and, under the Options tab in the palette, enter a title, description and ID for the option list. Make sure the title provides a clear indication of the contents of the list and the description is available to provide additional detail.
  4. Click the Add Option action to define the options. Text entry fields display for the order, value and display value for the first option.
  5. Enter an ACORD code (e.g., P) for the value.
  6. Enter the full description of the option that actually displays in the select list (e.g., Partial Improvement/Update) for the display value.
  7. Enter the order in which the values will display in the list (the first number should be 0).
  8. The values default as you add new options, but you can change them.
  9. Click the Used By tab in the palette to view a read-only list of transactions that use the particular option list, including the page, page element and field. Click on a transaction to go directly to the transaction.
