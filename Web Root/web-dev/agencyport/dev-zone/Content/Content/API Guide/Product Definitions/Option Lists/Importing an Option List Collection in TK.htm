Importing an Option List Collection via Toolkit

Perform the following to import a collection or option list via toolkit:

  1. Select the Lists folder in the navigation window.
  2. Click on the Import action in the editor. A pop-up window displays, prompting you to search for an option list collection file.

    NOTE:

    The only supported file type for an option list collection is .xml.

  3. Click on Browse to find and select the file.
  4. Enter a title for the list and an optional description, and click Create (or cancel this import action by clicking the X in the top right corner of the window). The new option list collection displays under the Lists folder.

    NOTE:

    If you import an option list collection, an option displays to indicate whether you want to override an existing option list file. If you do not select this option, and a duplication option list file exits, an error displays when you click Create.
