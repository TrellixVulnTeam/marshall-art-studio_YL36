Importing a View Collection via Toolkit

Perform the following to import views via toolkit:

  1. Select the Views folder in the navigation window.
  2. Click on the Import action in the editor. A pop-up window displays, prompting you to search for a views file.

    NOTE:

    The only supported file type for a views file is .xml.

  3. Click on Browse to find and select the file.
  4. Enter a title and an optional description for the list of views, and click on Create (or, cancel the import action by clicking the X in the top right corner of the window). The new ruleset displays under the Behaviors folder.

    NOTE:

    If you import a group of views, an option displays to indicate whether you want to override an existing views file. If you do not select this option and a duplicate views file exists, an error displays when you click Create.
