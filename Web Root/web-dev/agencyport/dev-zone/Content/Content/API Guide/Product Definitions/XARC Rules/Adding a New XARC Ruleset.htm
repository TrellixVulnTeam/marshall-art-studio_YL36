Adding a New XARC Ruleset

Perform the following to add a new XARC ruleset via toolkit:

  1. Select the Rules folder in the navigation window.
  2. Click the Add action in the editor. A pop-up window displays with some required data entry fields.
  3. Enter a file name, title and optional description for the rules collection and click Create (you can cancel this action by clicking the x in the top right corner of the window).

    NOTE:

    The file name must be unique. Select a name that will make the rules file easy to identify for maintenance and re-importing purposes.
