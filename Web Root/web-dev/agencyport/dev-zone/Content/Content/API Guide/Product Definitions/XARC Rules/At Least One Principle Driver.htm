At Least One Principle Driver

Rule:

Must be at least one principal driver on the quote.

Fields:

Driver Related to Applicant: PersAutoLineBusiness.PersDriver.PersDriverInfo.DriverRelationshipToApplicantCd

Approach:

If driver relation to application is 'Insured,' then she/he is the principal driver.

Should display a message when NOT(PersAutoLineBusiness.PersDriver.PersDriverInfo.DriverRelationshipToApplicantCd = ‘Insured’)

Steps to write this rule:

  1. Choose operand type of field and select the Relation to Applicant field.
  2. Choose "NOT =" for the operator.
  3. Choose operand type of value and enter IN (without the quotes) for the value.
  4. Enter the message to display.
