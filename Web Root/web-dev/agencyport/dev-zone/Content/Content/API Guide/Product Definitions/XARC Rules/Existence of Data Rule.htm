Existence of Data Rule

Rule:

Comprehensive glass deductible is mandatory when comprehensive deductible is selected for the vehicle.

Fields:

Comprehensive Deductible:

PersAutoLineBusiness.PersVeh.Coverage[CoverageCd='COMP'].Deductible[DeductibleTypeCd='FL' && DeductibleAppliesToCd='Coverage'].FormatInteger

Comprehensive Glass Deductible:

PersAutoLineBusiness.PersVeh.Coverage[CoverageCd='COMP'].Deductible[DeductibleTypeCd='FL' && DeductibleAppliesToCd='Glass'].FormatInteger

Make: PersAutoLineBusiness.PersVeh.Manufacturer

Model: PersAutoLineBusiness.PersVeh.VehIdentificationNumber

Year: PersAutoLineBusiness.PersVeh.ModelYear

Steps to write this rule:

  1. Add a condition to test for existence of comprehensive deductible by selecting the Comprehensive Deductible field for the left operand and exists operator. You don't have to select the right operand.
  2. Add condition to test if the Comp. Glass Deductible does not exist as shown below.
  3. Add variable for the vehicle data to be used in the message. Add message using these variables.
