Importing a XARC Ruleset

Perform the following to import a collection of XARC rules via toolkit:

  1. Select the Rules folder in the navigation window.
  2. Click the Import action in the editor. A pop-up window displays, prompting you to search for a rules file.

    NOTE:

    The only supported file type for a rules file is .xml.

  3. Click on Browse to search for and select a file.
  4. Enter a tile and optional description for the ruleset.
  5. Click Create (or cancel the import action by clicking the x in the top right corner of the window). The new ruleset displays under the Rules folder.

NOTE:

If you import a ruleset, an option displays to indicate whether you want to override an existing file. If this option is not selected, and a duplicate rules file exists, an error message displays when you click Create.
