Miscellaneous Rule - Example 1

Rule:

Create a declination rule around watercraft length. if watercraft length > 25 feet, display the following message: A boat length greater than 25 feet is not eligible for watercraft liability.

Fields:

Watercraft length: HomeLineBusiness.Watercraft.Length[UnitMeasurementCd='FT'].NumUnits

Steps to write this rule:

  1. Click on Add and pick condition to display a condition line. Click on where it says '(Select Field)' to select the watercraft length field. Under options in the palette, select Type; the available page, fieldset and field options display.
  2. Click on the operator and select > from the drop-down in the palette.
  3. Click on the second field. Select Value for the type and enter 25 for the value.
  4. Click on '(Please enter a message)' and enter the display message text. Select Error for the message severity since this is a validation rule. Select Yes for Show Duplicate Messages to display a message for each roster entry that meets the rule condition.
