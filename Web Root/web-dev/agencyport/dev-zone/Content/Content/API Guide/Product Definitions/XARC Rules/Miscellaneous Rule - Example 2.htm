Miscellaneous Rule - Example 2

Rule:

Create an underwriting rule around homeowners form and dwelling limit. If dwelling limit > 200,000 and HO2 or HO3 selected, display the following message: Due to the dwelling limit and HO form selected, this application will be referred to underwriting.

Fields:

Homeowners Form: HomeLineBusiness.Dwell[PrincipalUnitAtRiskInd='1'].PolicyTypeCd

Steps to write this rule:

  1. Add a condition for the rule.
  2. Define the operator for the condition.
  3. Add a similar condition for the dwelling limit.
  4. Add an underwriting message.

Additional Rule:

Add a step to the above rule to display the actual dwelling limit entered in the validation message (e.g., Due to the dwelling limit of ${Limit} and HO form selected, this application will be referred to underwriting).

Steps for writing this rule:

  1. Add a step to Rule #2 to display the dwelling limit in the validation message. Add a variable called Limit.
  2. Update the message to include the variable.
