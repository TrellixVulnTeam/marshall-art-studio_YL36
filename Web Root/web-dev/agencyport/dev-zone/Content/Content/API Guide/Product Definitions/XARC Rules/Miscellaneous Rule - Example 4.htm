Miscellaneous Rule - Example 4

Rule:

Create a declination rule around losses using the Sum Total function. if the amount of total losses entered on the Loss History page excess $15,000 over the last three years, display the following message: Due to the amount of losses entered, this submission will be declined.

Steps for writing the rule:

  1. Create a declination rule around losses using the Sum Total function. Click on a ruleset in the navigation and select Add Rule in the editor. Enter a name and description.
  2. Define the roster page source.
  3. Add a condition for the rule.
  4. Select the Sum Total function.
  5. Add the operator and the second part of the condition.
  6. Add another condition to define expression to check the last three years.
  7. Define the expression.
    1. Part 1
    2. Part 2
    3. Part 3 (DateAdd works as DateMinus when you use negative numbers)
  8. Define the operator.
  9. Add a message.
