Risk State Rule

Rule:

If the risk location state is either Florida, Georgia, South Carolina or North Carolina, display the following message: Since there is a hurricane warning out for your state, we are unable to write your policy at this time.

Fields:

Risk Location: Location.Addr.StateProvCd

Steps to write this rule:

  1. Add a condition and choose the left operand type of field and select the risk location state.
  2. Choose IS IN operator. For the right operand, choose the operand type of value and enter the values of state codes separated by commas.
  3. Add the appropriate message.
