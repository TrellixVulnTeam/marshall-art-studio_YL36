XARC Rules

XARC rules are created in the Rules section of the Toolkit. These rules can be any one of the following types and display on submit of a page:

  • Validation - This type of rule alters the user, after submitting a page, that they performed an invalid action. They not allowed to continue.
  • Warning - This type of rule provides an alert to the user, after submitting a page, that data they entered in the application requires review, but is allowed to continue.
  • Information - This type of rule provides some informational text, after submitting a page, based on an action performed by the user. They are allowed to continue.
