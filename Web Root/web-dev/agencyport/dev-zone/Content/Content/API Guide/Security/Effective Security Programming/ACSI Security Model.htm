Fleshing Out the ACSI Security Model

The ACSI security model typically varies very little from project to project. In fact, it is our believe that the default Java classes for the security model entities found in the com.agencyport.security.model.impl package can be used out of the box in many cases. However, there may be a situation when your application may require that one or more of these entities need an additional property in order to fulfill a specific business requirement set forth by your customer.
