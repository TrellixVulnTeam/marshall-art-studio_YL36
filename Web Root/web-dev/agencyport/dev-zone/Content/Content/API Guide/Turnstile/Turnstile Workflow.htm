Turnstile Workflow

Unfortunately, many agents still attach ACORD forms as PDFs and email them to carriers. This involves a burdensome, manual re-keying process of those forms for carriers into their point-of-sale and back end systems, which can result in data errors, slow response time and high costs.

This is where Turnstile, a clean, simple and ultra-secure service included in AgencyPortal 5.0, comes in. Turnstile is a subscription carrier web service that takes in ACORD forms (in PDF, TIFF and ZIP format) and returns ACORD XML to feed into point-of-sale, CRM or policy administration systems. This functionality shortens intake time, accelerates decision making, reduces processing cost and eliminates lost opportunities.

Turnstile also provides faster turnaround, which results in more business.

Prefill from ACORD Upload

The Prefill from ACORD Upload option on the Get Started menu in AgencyPortal 5.0 allows you to upload a .pdf, .tif, .tiff or .zip file of ACORD form(s) into AgencyPortal.
