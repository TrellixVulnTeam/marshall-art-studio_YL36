Downloading All Dependencies to a Local Location (to Support ANT)

Perform the following to download all dependencies to a local location:

  1. Add the following configuration into your POM file and change the output directory to one you want to use. Refer to the Copying project dependencies topic on the Maven site for more information.
    This creates a flat list of jars and war dependencies used by the POM file.
  2. Open a command line and run the following to clean out the target folder:

    3. >mvn clean

  3. Run the following command to dump all of the project's runtime dependencies (JAR and WAR files) into the target/dependency folder.

    4. mvn dependency:copy-dependencies -DincludeScope=runtime -P !default

    NOTE:

    Note how we explicitly deactivate the 'default' profile. This is because this profile brings in some Tomcat specific dependencies, which are present on most other containers.
