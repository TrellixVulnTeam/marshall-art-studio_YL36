Importing the Project into Eclipse

Perform the following to import the project into Eclipse as a Maven project if the project has already been checked out of SVN. This installs a template root along with the pom.xml that holds the dependencies to AgencyPortal SDK.

  1. Right click on the project and select the Import menu.
  2. Click the Import option.
  3. Select Existing Maven Projects and click Next.
  4. Click the Browse button to browse to the location where you created your project (this is the location you used earlier with the command line prompts in the Setting Up Your Portal 5 Project topic).
  5. Check the check box next to the project and click Finish.

    NOTE:

    The productDatabase.xml file is the comprehensive version with all LOBs included. Disable the ones you are not interested in. The web.xml file includes all servlets; comment out the ones you do not plan to use.
