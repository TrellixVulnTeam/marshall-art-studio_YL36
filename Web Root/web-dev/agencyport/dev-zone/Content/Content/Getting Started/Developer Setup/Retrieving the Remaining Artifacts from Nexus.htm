Retrieving the Remaining Artifacts from Nexus

Perform the following to download the SQL and SOLR home artifacts directly from the Nexus repository:

  1. Browse directly to the Nexus server: http://10.21.39.108:8081/nexus/#view-repositories.
  2. Navigate to either the Releases or Snapshots section, depending on your needs.
  3. Perform the following:
    SQL
    1. Download the SQL zip file from the Nexus server and extract it.
    2. Run the create scripts per your database. Refer to the Deploying Project Locally via Maven section for information pertaining to JNDI configuration.

    SOLR Installation

    4. NOTE:

    This is a developer setting; production setup may vary.
    1. Download Solr from the Nexus server. The Solr zip is located under com/Agencyport/sdk/(your_desired_version)/apwebapp-<your_desired_version>-solr.zip.
    2. Set up SOLR home in your application properties file as follows:
      solr_home=workspace/runtime/AgencyPortal/SOLR_HOME
