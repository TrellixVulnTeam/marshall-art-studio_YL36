Setting Up a Tomcat Server

Perform the following to set up a new server and point it to a local Tomcat instance:

  1. Make sure your local Tomcat installation has the necessary JDBC jars made available.
  2. Set up a new server within Eclipse and point it to your Tomcat installation:
    1. Right click on the Servers view of the Java EE perspective.
    2. Select New.
    3. Click Server.
  3. Select the Tomcat v7.0 Server and click Finish.

    NOTE:

    Do NOT add any modules yet.


    Do not edit your runtime environments at this time. Just make sure your Tomcat and Eclipse project are using JDK 7.
    You can return and edit your runtime environments at a later time by navigating to Window > Preferences > Server > Runtime Environments.
  4. In the server settings, select the Use Tomcat installation (takes control of Tomcat installation) option and change the Deploy path to webapps.
  5. Increase the memory by adding -Xmx2G to the VM arguments.
  6. Add your newly created project to the server by highlighting an available resource and clicking the Add button.
  7. Disable the Auto Reload for the module.
  8. Click Finish.
