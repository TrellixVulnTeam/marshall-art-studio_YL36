WebSphere 8.5

Preparing a WebSphere 8.5 Environment for an AgencyPortal Deployment

Since AgencyPortal is packaged as a J2EE compliant web application, it is compatible with any J2EE compliant web container. However, in order to get the application up and running in WebSphere, we must take some steps to ensure that the WAR/EAR file and the container are properly configured. This is due to some of the characteristics of WebSphere, such as its default classloading policies, default bootstrap process, and bundled IBM JDK.

Refer to the Requirements topic for a comprehensive list of the changes that will need to be made.
