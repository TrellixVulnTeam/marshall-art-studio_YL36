My Accounts

When you click the My Accounts tab, all accounts display.

And/or display accounts by entering account information in the search box.

Accounts filter as soon as a letter is typed into the search box. Active filtering searches the following:

  • first name
  • last name (personal lines)
  • company name (commercial lines)

Backspacing in the search box re-engages active filtering and displays results based on the value in the search box.

NOTE:

The basic search box uses "contains" logic and is not case sensitive. However, you must enter the exact string of characters, including symbols. For example, a search for ABC will find ABC Industries, but not A.B.C. Industries.

The search feature also uses auto suggest when three characters are entered into the search box.

  • Auto suggest returns a maximum of four matches. If there are more than four matches, XX more (where XX is the amount of additional items that meet the search criteria) displays to link additional results.
  • The XX more link is informational only. If selected, the auto suggest results no longer display.

The search criteria remains in the search box, regardless of what action you perform. This includes logging out.

Remove the search criteria from the search box to unfilter the results.
